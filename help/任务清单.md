## 不变约束

- 后续训练、推理、评测、benchmark 和报告必须显式绑定 tokenizer 元数据。
- 禁止混用 GLM5.1 checkpoint、DS tokenizer、DS checkpoint、GLM5.1 tokenizer。
- 后续开发、训练、推理均以 DS tokenizer 为标准，不再兼容旧 tokenizer。
- checkpoint loader 只接受当前 schema，不为旧 `.pth` 文件放宽校验。
- 模型结构必须通过 `ModelConfig` 或 checkpoint 内的 `model_config` 快照恢复。
- LongRoPE2 候选因子 sweep 只能临时覆盖评测进程内配置，不得写回 checkpoint 或训练产物。
- `CUDA_VISIBLE_DEVICES` 是单机多卡资源池边界；程序内部 device map 只使用 PyTorch 暴露后的逻辑设备号，例如 `cuda:0 / cuda:1`。
